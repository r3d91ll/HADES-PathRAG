HADES-PathRAG
Enhanced PathRAG implementation with XnX notation, ArangoDB and Ollama integration

⚡ Quick Start

Get up and running with HADES-PathRAG in minutes:

  1. Clone the repository: git clone https://github.com/r3d91ll/HADES-PathRAG.git
  2. Install dependencies: pip install -e .
  3. Set up Ollama: ./scripts/setup_ollama.sh
  4. Follow our Getting Started Guide
